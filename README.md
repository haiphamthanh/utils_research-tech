# Research Tech

Hệ thống quản lý dữ liệu học tập, tìm kiếm ngữ nghĩa và trả lời câu hỏi học thuật dựa trên tài liệu. Mục tiêu là tạo bài học ngắn gọn, có cơ sở và tập trung vào kỹ năng người học cần phát triển.

## Chạy local

```bash
./start.sh
./stop.sh
```

Mặc định website chạy tại `http://127.0.0.1:5200`. Có thể thay đổi bằng biến môi trường `HOST` và `PORT`.
